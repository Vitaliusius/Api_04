# Космос в Telegram

Загружает фотографии космоса и запуска ракет и сохраняет их, а также постит их в Telegram-канале.

## fetch_nasa_apods.py

Скрипт скачивает фотографии дня NASA (apod) в папку "images", если папки с таким названием нет в директории скрипта, она будет создана автоматически.

### Запуск скрипта

Потребуется API ключ NASA. Получить его можно [здесь](https://api.nasa.gov), затем его нужно присвоить переменной окружения NASA_API_KEY в файле'.env',  как в примере ниже:

```
NASA_API_KEY=hfgj5kdkhjkj5kjhf78
```

Скрипт запускается командой:

```
python fetch_nasa_apods.py
```

## fetch_spacex_launches.py

Скрипт скачивает фотографии запусков SpaceX в папку "images", если папки с таким названием нет в директории скрипта, она будет создана автоматически. По умолчанию скрипт скачивает последний запуск.

### Запуск скрипта

Скрипт запускается командой:

```
fetch_spacex_launches.py
```

Для загрузки фотографий определенного запуска, нужно указать его номер в качестве аргумента как в примере ниже:

```
python fetch_spacex_launches.py 25
```

## posting_telegram_bot.py

Скрипт бесконечно постит скачанные фотографии в телеграмм канал, пока запущен. Сначала постит по порядку все фотографии, затем пермешивает их, снова постит, снова пермешивает, снова постит и т.д.

### Запуск скрипта

Вам потребуется API token вашего бота телеграмм, а так же chat id вашего Telegram-канала. Так же нужно сделать вашего бота администраторм вашего канала. API token вашего бота присвоить переменной POSTING_TELEGRAM_BOT_API_KEY, chat id вашего канала присвоить переменной TELEGRAM_CHAT_ID в файле '.env' как в примере ниже:

```
POSTING_TELEGRAM_BOT_API_KEY=jf85heik58fydjdkfifundk7
TELEGRAM_CHAT_ID=@channel_id
```

По умолчанию интервал между постами составляет 4 часа, скрипт запускается командой:

```
posting_telegram_bot.py
```

Чтобы поменять интервал между постами, необходимо указать количество часов в качестве аргумента, как в примере ниже:

```
posting_telegram_bot.py 1
```

## post_photo.py

Скрипт постит одну фотографию в Telegram-канал.

### Запуск скрипта

Вам потребуется API token вашего бота телеграмм, а так же chat id вашего Telegram-канала. Так же нужно сделать вашего бота администраторм вашего канала. API token вашего бота присвоить переменной POSTING_TELEGRAM_BOT_API_KEY, chat id вашего канала присвоить переменной TELEGRAM_CHAT_ID в файле '.env' как в примере ниже:

```
POSTING_TELEGRAM_BOT_API_KEY=jf85heik58fydjdkfifundk7
TELEGRAM_CHAT_ID=@channel_id
```

Скрипт по умолчанию постит рандомно выбранную фотографию и запускается командой:

```
post_photo.py
```

Для поста конкретной фотографии необходимо ввести ее имя после команды в качестве аргумента, как в примере ниже:

```
post_photo.py nasa_apod_1.jpg
```

pip install -r requirements.txt
